# Enigma

## How to run it
1) `chmod +x go.sh`
2) `./go.sh`
3) Enjoy

TESTED ON ARCH LINUX, 07.04.2023

## Content
- `backend` contains all code related to the websocket
- `frontend` contains the config & static data for the frontend
Use the built-in continuous integration in GitLab.
